# Triple Space Translator 发布文案

![Release 封面图](../assets/marketing/release-cover.png)

## 标题

三连空格，中文英文原位切换。

## 简介

Triple Space Translator 允许你先输入中文或英文，再在 `0.5 秒` 内连按 `3 次空格`，当前文本就会直接原位翻译并替换。再次三连空格，可以切回上一轮语言。

## 为什么它值得下载

- 保留你最自然的母语思考方式。
- 在需要和国外 AI 大模型交流时，再瞬间切成英文。
- 始终停留在同一个聊天框、搜索框或编辑器里。
- 省去复制、粘贴、打开翻译器、再贴回来的来回操作。

## 当前平台状态

- macOS：已可用，使用 Apple `Translation.framework`
- Windows：稳定版当前支持在线 API 翻译
- Windows 离线内置模型：仍在持续完善中

## 适合的场景

- ChatGPT、Claude、Grok、Gemini
- 浏览器搜索框
- 聊天回复和即时输入
- 提示词撰写和短文档草稿
